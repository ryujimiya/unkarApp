削除依頼

削除依頼について

unkarは２ちゃんねるへの書き込みを勝手に保存しているサイトです。

書き込み情報の削除をご希望の方は、削除を希望するページのURLと書き込み番号（任意）を添えて、管理人へメールしてください。
削除依頼のメールへの返信は基本的に行っておりません。ご了承ください。

注意事項

  • 削除が行われたか確認する場合、ブラウザのキャッシュを削除してから行ってください。
    ブラウザのキャッシュを削除しなければ、unkarから情報が削除されている場合でも表示されてしまいます。
  • 2ちゃんねるで削除されていない書き込み情報の場合、上手く削除できない場合があります。
    2ちゃんねるで削除されていない書き込み情報を削除したい場合、2ちゃんねるで削除されていないことを明記してご連絡いただけると対応が多少早くなります。
  • 2週間以上経過しても対応されていない場合、再度ご連絡いただけると大変助かります。
  • 削除対応可能な情報は、unkar.org、unkar.jpまたwww.heiwaboke.netに保存されているスレッドに限ります。
